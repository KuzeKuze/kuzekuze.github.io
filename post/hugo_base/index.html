用hugo建立自己的博客

May 15, 2020

使用hugo建立属于自己的博客。

第一步：建立站点

创建博客命令：

$ hugo new site blog_path
Congratulations! Your new Hugo site is created in C:\Users\xxx\xxx_blog.

Just a few more steps and you're ready to go:

1. Download a theme into the same-named folder.
   Choose a theme from https://themes.gohugo.io/ or
   create your own with the "hugo new theme <THEMENAME>" command.
2. Perhaps you want to add some content. You can add single files
   with "hugo new <SECTIONNAME>\<FILENAME>.<FORMAT>".
3. Start the built-in live server via "hugo server".

Visit https://gohugo.io/ for quickstart guide and full documentation.

执行该命令后会在blog_path路径下创建博客文件夹。我们查看一下该文件夹里的东西。

第二步：添加主题

我们已经建立好了博客，接下来设置主题。我们可以在themes寻找自己喜欢的主题。

每个主题下方会有安装该主题的教程。

我们按部就班的按照教程中的方法安装这个主题。注意主题一般提供了不同的方法应对不同的情况

首先要确定自己在博客文件的根目录下。

$ cd themes
$ git clone https://github.com/marcanuy/simpleit-hugo-theme

我们安装好主题后，先确保自己在博客的根目录下，而不是在themes目录下，在本地启动博客并查看(manis是主题的名称)：

$ hugo server -t manis --buildDrafts

下一步我们新建文章，所以这里用ctrl+c退出一下。

第三步：添加文章

文章是markdown类型的。以下命令在博客根目录的content/post目录下建立一个.md笔记。

$ hugo new post/blog.md
C:\Users\duluomin\myblog\content\post\blog.md created

接下来可以编辑blog.md内容。注意最开头的是标题。

然后我们再次启动博客，可以看见我们的内容。

点击我们就可以看到博客内容。

第四步：建立blog仓库

在github中建立一个仓库，仓库名字为你自己在github的id的小写加上**.github.io**，比如：

我在github中的id为KuzeKuze，那么我的仓库名称应该是kuzekuze.github.io

你的仓库应该是一个空仓库，我的这个已经提交过一次了。

仓库建立好了过后我们来进行部署。

第五步：将个人博客部署到远端服务器

先确保自己在博客根目录下。键入以下命令：

hugo --theme=manis --baseUrl="https://kuzekuze.github.io/" --buildDrafts

此时会在博客根目录下生成一个public文件，我们现在要做的就是将这个文件夹的内容传到我们刚刚建立好的仓库中。我们首先将public文件夹初始化为本地仓库：

$ git init
$ git add *
$ git commit -m "我的hugo第一次提交"

接下来将这个仓库和远端的仓库进行关联。

$ git remote add origin https://github.com/KuzeKuze/kuzekuze.github.io

我们现在将这个仓库推到远端：

$ git push

这一步有可能会出错。
